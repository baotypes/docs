#### we no longer use the [daos.fun](http://daos.fun) wallet browser extension

- this browser extension is no longer supported. 
- the [daos.fun](http://daos.fun) wallet browser extension is deprecated.
- you can uninstall this browser extension from your browser. 
- if you used to use this browser extension, you have to resetup your admin panel.

#### setting everything up to trade

- go to your admin panel while you are logged in with your Creator wallet. 
- in the Linked Wallets section, click "manually add delegate"
- paste in your wallet address 
- sign the transaction when prompted
- great\! now you've linked your wallet.
